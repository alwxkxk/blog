# 个人博客
- 生子当如哈士奇的个人博客，包含若干技术文章，以及《软硬结合——从零打造物联网》、《3D可视化教程》两个系列教程。

## 运行
本博客基于hexo搭建：

```bash
cd hexo
npm install hexo -g
npm install -g hexo-cli
npm install

# 运行
hexo s
```
