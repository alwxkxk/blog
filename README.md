# 个人博客
- 生子当如哈士奇的个人博客，包含若干技术文章，以及《软硬结合——从零打造物联网》、《3D可视化教程》两个系列教程。
## README
个人博客（基于hexo）
```bash
npm install hexo -g
npm install -g hexo-cli
npm install
hexo new 新文章标题
hexo generate // hexo V4.2.1 以上才兼容nodeV14，否则生成空白文件。

# 运行
hexo s
```
